# Tools

## macOS/Linux

* `bootstrap.sh` — install/link dotfiles into `$HOME`
* `update.sh` — git update (`pull --rebase --autostash`)

## Windows

* `windows/setup.ps1` — Windows installer (gVim + git config)
* `windows/_vimrc` / `windows/_gvimrc` — gVim entry rc files

See `../CROSS_PLATFORM.md`.
